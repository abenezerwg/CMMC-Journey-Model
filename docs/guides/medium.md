# CMMC Journey Path: Medium Enterprise (51–250 employees)

⚠️ **Disclaimer:** This is an unofficial, open-source guide. Use for educational purposes only.

## Overview
- Full scope (all CUI systems, multiple departments)
- C3PAO required
- 6-month timeline
- ~$50K budget
- Tier 1 + Tier 2 fully, Tier 3 controls prioritized

## Recommended Stages

### 1. Discover
- Business case, contract obligations
- Executive buy-in
- ROI and resource planning

### 2. Scope
- Asset inventory
- CUI/FCI classification
- Boundary definition
- Enclave decision if multiple systems

### 3. Assess
- Detailed gap analysis across all required controls
- POA&M planning
- Control prioritization (Tier 1 → Tier 3)

### 4. Implement
- Critical controls first
- Evidence collection
- Policy documentation and staff training

### 5. Document
- Complete SSP covering all systems
- Finalized POA&M
- Evidence package ready for C3PAO

### 6. Certify
- C3PAO selection
- Pre-assessment review
- Formal assessment and continuous compliance

## Tips
- Use structured project management tools
- Ensure executive sponsorship
- Plan for resource allocation across departments
